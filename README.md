# نظام تصنيف السير الذاتية

# عن المشروع 
هذا المشروع يهدف إلى بناء نموذج برمجي يقوم بتحليل وتصنيف ملفات السير الذاتية (CVs) تلقائياً لتسهيل عملية التوظيف، وذلك باستخدام تقنيات معالجة اللغات الطبيعية (NLP).

# التقنيات المخطط استخدامها 
* Language: Python
* Library: NLTK / Scikit-learn
* Tools: Pandas for data handling

# خطة العمل 
- جمع وتجهيز قاعدة بيانات للسير الذاتية (Data Collection).
- مرحلة تنظيف النصوص (Text Preprocessing).
- تدريب النموذج (Model Training).
- اختبار الدقة (Testing & Evaluation).
